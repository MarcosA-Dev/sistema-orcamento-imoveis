# Projeto: Sistema de Orçamento de Imóveis 

Programa simples em **Python** que roda direto no **terminal**, feito para ajudar no cálculo e gerenciamento de orçamentos de imóveis.  
O foco é praticar **lógica de programação, funções e classes**, mantendo o código organizado e fácil de entender.

---

## Como usar
1. Tenha o **Python 3** instalado.  
2. Baixe os arquivos do projeto.  
3. No terminal, entre na pasta e execute:

   ```bash
   python main.py
